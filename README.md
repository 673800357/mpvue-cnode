# mpvue-cnode

> 使用mp-vue开发的微信小程序版的cnode。

API来自于cnode社区

想熟悉下vue的开发流程，又想体验下mpvue，于是写了个练手项目

## Build Setup

``` bash
# install dependencies
npm install

开发模式
```
npm run dev
```


目录结构
- src
  - pages
    - index 主页
    - detail 帖子详情页
    - me 我的主页
    - publisg 发帖页
    - notice 通知页
    - login 登录页
    - user 用户主页
  - components
    - card
